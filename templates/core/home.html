O projektu

ChemLab Manager je informacioni sistem dizajniran za istrazivače u hemiji. Omogucava upravljanje jedinjenjima, literaturom, eksperimentima i inventarom.

  • 3D prikaz hemijskih struktura
  • CRUD operacije
  • REST API za integraciju
  • Autentifikacija i autorizacija
  • Pracenje inventara
  • Beleske i komentari
